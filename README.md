SetUp :
----------------
1) Download and install node.js
     
     chec: node -v
            npm -v

2) Install angular

    npm install -g @angular/cli@15

    check : ng version

3) create new app

    cd desktop
    ng new my-app

4) start the server

    cd desktop
    ng new my-app

5) start the server

    ng serve

    check: in browser localhost:4200

**) allow scripts execution ffrom powershell

    2) run command : set-executionPolicy unrestricted
